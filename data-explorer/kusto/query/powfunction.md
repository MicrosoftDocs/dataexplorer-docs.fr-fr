# <a name="pow"></a>pow()

Retourne le résultat de l’élévation au pouvoir

**Syntaxe**

`pow(`*exposant de base* `,` *exponent*`)`

**Arguments**

* *base*: Valeur de base.
* *exposant*: Valeur exposante.

**Retourne**

Base de retours soulevée à l’exposant de puissance : exposant de base.
