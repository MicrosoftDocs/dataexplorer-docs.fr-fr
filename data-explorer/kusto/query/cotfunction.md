# <a name="cot"></a>cot()

Calcule la cotangente trigonométrique de l’angle spécifié, en radians.

## <a name="syntax"></a>Syntaxe

`cot(`*x*`)`

## <a name="arguments"></a>Arguments

* *x*: nombre réel.

## <a name="returns"></a>Retours

* La valeur de fonction cotangente pour `x`
