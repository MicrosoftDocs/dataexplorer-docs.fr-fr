# <a name="binary_not"></a>binary_not()

Retourne une négation un peu sage de la valeur d’entrée.

```kusto
binary_not(x)
```

**Syntaxe**

`binary_not(`*num1*`)`

**Arguments**

* *num1*: numérique 

**Retourne**

Retourne logique PAS de fonctionnement sur un nombre: num1.
