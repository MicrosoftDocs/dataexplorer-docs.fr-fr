# <a name="abs"></a>abs()

Calcule la valeur absolue de l’entrée.  

**Syntaxe**

`abs(`*X*`)`

**Arguments**

* *x*: Un intégrant ou un nombre réel, ou une valeur de timespan.

**Retourne**

* Valeur absolue de x.
