# <a name="abs"></a>abs()

Calcule la valeur absolue de l’entrée.  

## <a name="syntax"></a>Syntaxe

`abs(`*x*`)`

## <a name="arguments"></a>Arguments

* *x*: nombre entier ou réel, ou valeur TimeSpan.

## <a name="returns"></a>Retours

* Valeur absolue de x.
