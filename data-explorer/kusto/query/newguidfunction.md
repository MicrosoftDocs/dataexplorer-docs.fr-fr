# <a name="new_guid"></a>new_guid()

Retourne un GUID aléatoire (identificateur global unique).

```kusto
new_guid()
```

## <a name="syntax"></a>Syntaxe

* `new_guid()`-génère une nouvelle valeur de type `guid` .
